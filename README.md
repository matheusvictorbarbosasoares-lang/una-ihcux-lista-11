# ReCiclo App

## Integrantes
Matheus Victor

## Proposta de valor
O aplicativo ReCiclo ajuda pessoas a doar e encontrar itens usados, promovendo sustentabilidade.

## Telas
- Home: lista de itens disponíveis
- Filtros: permite buscar por categoria e distância
- Detalhes: mostra informações do item
- Cadastro: permite cadastrar um item
- Chat: conversa entre usuários
- Perfil: mostra conquistas e histórico

## Jornada do usuário
O usuário entra no app, visualiza itens, escolhe um item, entra em contato via chat e combina a retirada.
